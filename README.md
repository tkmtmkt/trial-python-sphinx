trial-python-sphinx
===================

環境設定
--------

オンライン環境でインストールし、wheelパッケージをダウンロードする。

```sh
$ ./python/install.sh
```

オフライン環境でwheelパッケージからインストールする。

```sh
$ ./python/offline-install.sh
```


実行
----

作業用シェルを起動する。

```sh
$ ./bin/shell.sh
```

```sh
$ sphinx-quickstart
```
